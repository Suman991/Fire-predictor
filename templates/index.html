Algerian Forest Fire Predictor

Introduction

Welcome to the Algerian Forest Fire Predictor application. This tool helps predict the Forest Fire Weather Index (FWI), which is a crucial measure in assessing the likelihood and severity of forest fires. The application uses several key parameters to provide an accurate prediction.

About the Parameters

  • Temperature: Air temperature in Celsius.
  • RH: Relative Humidity in percentage.
  • Ws: Wind speed in km/h.
  • Rain: Amount of rainfall in mm.
  • FFMC: Fine Fuel Moisture Code - Indicates surface litter moisture.
  • DMC: Duff Moisture Code - Indicates moisture of moderately deep organic layers.
  • ISI: Initial Spread Index - Represents fire spread potential.
  • Classes: Fire severity classes or fire category levels.
  • Region: Location-based forest fire region.
  • FWI: Final Forest Weather Index value.
